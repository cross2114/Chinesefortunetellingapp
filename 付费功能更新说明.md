## 🔐 数据安全

### LocalStorage结构

```json
{
  "mysticArtsUser": {
    "id": "abc123",
    "email": "user@example.com",
    "name": "John",
    "subscription": "premium",
    "joinedDate": "2026-03-30T10:00:00.000Z",
    "trialEndsAt": "2026-04-06T10:00:00.000Z",
    "subscriptionStartedAt": null,
    "readingsUsed": {
      "bazi": 0,
      "iching": 0,
      "tarot": 0,
      "face": 0
    }
  }
}
```

**状态判断**：
- `trialEndsAt` 存在 + 未过期 + 无 `subscriptionStartedAt` = 试用中
- `subscriptionStartedAt` 存在 = 付费订阅中
- 两者都不存在 = Free计划

   - 使用React DevTools查看Context

3. **常见问题**：
   - Q: 横幅不显示？
     A: 检查用户是否在试用期，是否已关闭
   
   - Q: 付费墙不弹出？
     A: 检查`canUseReading`返回值
   
   - Q: 时间计算错误？
     A: 检查时区设置，使用ISO字符串
