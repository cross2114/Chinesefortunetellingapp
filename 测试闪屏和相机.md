# 🧪 测试闪屏和相机功能

## 📋 准备工作

### **1. 在Xcode中添加相机权限**

参考错误提示，在 `Info.plist` 中添加：

| Key | Value |
|-----|-------|
| `Privacy - Camera Usage Description` | `We need camera access to analyze your facial features for fortune reading` |
| `Privacy - Photo Library Usage Description` | `We need photo library access to let you upload photos for face reading` |
| `Privacy - Photo Library Additions Usage Description` | `We need access to save photos for your face reading records` |

### **2. 配置LaunchScreen（闪屏）**

详细步骤见：`/iOS闪屏配置指南.md`

**快速版：**
1. 打开 `LaunchScreen.storyboard`
2. 删除所有UI元素
3. 设置背景色为 `#0F0A08`
4. 保存

---

## 🚀 构建和运行

```bash
# 1. 重新构建前端
npm run build

# 2. 同步到iOS项目
npx cap sync ios

# 3. 打开Xcode
npx cap open ios
```

在Xcode中：

1. **Clean Build Folder** (Cmd + Shift + K)
2. **删除模拟器/真机上的旧APP**
3. **重新运行** (Cmd + R)

## 🐛 常见问题

### **问题1：看不到闪屏动画**

**可能原因：**
- 原生LaunchScreen还在显示
- React还没加载完成

**解决方案：**
1. 确认已配置 `LaunchScreen.storyboard`（见指南）
2. 确认 `capacitor.config.ts` 中 `launchShowDuration: 0`
3. 删除APP重新安装

---

### **问题2：相机崩溃**

**错误信息：**
```
This app has crashed because it attempted to access 
privacy-sensitive data without a usage description
```

**解决方案：**
在Xcode的 `Info.plist` 中添加相机权限描述（见上方）

---

### **问题3：首页布局太拥挤或太空**

**解决方案：**
已优化移动端间距：
- Logo: 32px
- 标题: 16px
- 卡片最大高度: 500px
- 顶部padding: 16px
- 底部padding: 自动

如果还有问题，截图给我看！

---

### **问题4：闪屏一闪而过**

**解决方案：**
已设置持续时间为 **3500ms (3.5秒)**

如果还是太快，可以在 `/src/app/App.tsx` 中修改：
```typescript
duration={5000} // 改为5秒
```
