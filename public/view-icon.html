✨图标设计说明

风格：敦煌壁画美学 + 现代神秘主义

主要元素：汉字"玄"、太极阴阳、八卦八角形

配色：赭石红、土黄金、石青蓝、深色背景

尺寸：1024 x 1024 像素

📱下一步：生成所有尺寸

  1. 右键点击上面的图标 → 选择"将图像另存为..."
  2. 保存到桌面，文件名改为：icon-1024.png
  3. 点击下面的按钮访问AppIcon.co
  4. 上传刚才保存的PNG文件
  5. 勾选 ☑️ iPhone 和 ☑️ iPad
  6. 点击 Generate 生成
  7. 下载生成的zip文件
  8. 解压得到 AppIcon.appiconset 文件夹
🚀打开 AppIcon.co 🔄在线转换 SVG → PNG

💡提示：如果右键保存不行，可以截图：

1. 按 Cmd + Shift + 4

2. 按 空格键（光标变成相机）

3. 点击这个浏览器窗口

4. 截图自动保存到桌面
