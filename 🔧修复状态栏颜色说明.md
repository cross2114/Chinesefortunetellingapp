# 🔧 状态栏颜色修复说明

## ✅ 已完成的修复

### 问题
应用打开后，手机顶部状态栏背景是白色的，与深色敦煌美学设计不匹配。

### 原因
- `manifest.json` 中的 `theme_color` 设置为金色 `#D4A76A`
- 缺少 HTML `<meta name="theme-color">` 标签
- 缺少 iOS 专用的状态栏配置

### 修复内容

#### 1. 创建了 `/index.html` 文件
添加了完整的 meta 标签配置：

```html
<!-- 通用状态栏颜色 -->
<meta name="theme-color" content="#0f0a08">

<!-- iOS 专用配置 -->
<meta name="apple-mobile-web-app-status-bar-style" content="black-translucent">
```

#### 2. 更新了 `/public/manifest.json`
```json
{
  "theme_color": "#0f0a08"  // 从金色改为深色背景
}
```

#### 3. 创建了 `/src/main.tsx`
React应用的入口文件，确保应用正确启动。

## 💡 技术细节

### theme-color 优先级

1. **HTML meta标签**（优先级最高）
   ```html
   <meta name="theme-color" content="#0f0a08">
   ```

2. **Manifest.json**
   ```json
   "theme_color": "#0f0a08"
   ```

3. 如果HTML中有meta标签，会覆盖manifest中的设置

### iOS 状态栏样式

有三个选项：
- `default`: 黑色文字（适合浅色背景）
- `black`: 黑色文字（适合浅色背景）
- `black-translucent`: 白色文字，半透明背景（适合深色背景）✅ 我们用的这个
