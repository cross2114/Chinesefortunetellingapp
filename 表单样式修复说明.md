## 🔍 密码输入框特别优化

### 视觉效果对比

#### 普通文本输入
```
┌─────────────────────┐
│ hello world         │  ← 默认字体
└─────────────────────┘
```

#### 密码输入（新样式）✅
```
┌─────────────────────┐
│ • • • • • • •       │  ← 等宽字体，间距更宽
└─────────────────────┘
```

### 技术实现
```tsx
// 根据input type动态添加样式
type === "password" && "tracking-wider font-mono"
```

**优点**：
- `font-mono`：等宽字体，每个点点占用相同宽度
- `tracking-wider`：字母间距增加，点点之间更清晰
- 提升密码长度的可视化辨识

## 💡 设计理念

### 1. 渐进式披露
- 占位符提供刚好够用的信息
- 不过度说明，不过度简化
- 在输入框内提供即时指导

### 2. 视觉层次
```
优先级从高到低：
1. 用户输入的内容（100%不透明）
2. Label标签（正常不透明度）
3. 占位符（60%透明度）
4. 边框和背景（更淡）
```

### 3. 反馈即时性
- 聚焦：边框高亮 + 光晕
- 输入：占位符消失
- 错误：红色边框 + 提示
- 成功：（可选）绿色标记
