# ⚡ 一分钟看懂怎么装到手机

## 🎯 目标
把你的应用装到手机上，像微信、抖音那样用

---

## 📋 三步走

### 1️⃣ 上传代码（GitHub）
```
你的电脑 → GitHub网站
```
- 用GitHub Desktop拖拽上传
- 或者：https://github.com/new 手动创建

### 2️⃣ 发布网站（Vercel）
```
GitHub → Vercel → 得到网址
```
- 登录Vercel.com
- 点Import → 选项目 → Deploy
- 拿到网址：`https://你的应用.vercel.app`

### 3️⃣ 安装到手机
```
手机浏览器 → 打开网址 → 点安装
```
- **安卓**：Chrome打开 → 点"安装"
- **苹果**：Safari打开 → 分享 → 添加到主屏幕

---

## 💡 核心概念

### 为什么要GitHub？
- Vercel需要从某个地方读取代码
- GitHub就是代码的"云盘"

### 为什么要Vercel？
- 免费把你的代码变成网站
- 自动提供HTTPS（PWA必须的）

### 为什么要HTTPS？
- PWA（可安装的网页应用）必须用HTTPS
- Vercel自动提供，你不用管

## 💬 常见对话

**问**：为什么不能直接装？
**答**：因为还在你电脑上，手机访问不了。要先放到网上。

**问**：Vercel要钱吗？
**答**：免费！个人用足够了。

**问**：以后怎么更新？
**答**：改代码 → 上传GitHub → Vercel自动更新。用户打开会自动获取新版本。

**问**：可以改网址吗？
**答**：可以！在Vercel设置里绑定自己的域名。

**问**：会不会被别人看到代码？
**答**：GitHub可以设置私有仓库。但网站是公开的（任何人都能访问）。
