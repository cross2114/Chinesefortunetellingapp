## 🛠️ 技术配置

### Capacitor 安装
```bash
# 1. 安装Capacitor
npm install @capacitor/core @capacitor/cli
npm install @capacitor/ios @capacitor/android

# 2. 初始化（按提示填写）
npx cap init

# 3. 构建Web应用
npm run build

# 4. 添加iOS平台
npx cap add ios

# 5. 同步文件
npx cap sync ios

# 6. 打开Xcode
npx cap open ios
```

- [ ] 完成Capacitor安装
- [ ] 配置`capacitor.config.ts`
- [ ] 设置正确的Bundle ID（如：`com.yourcompany.mysticarts`）

## 🆘 遇到问题？

### 常见问题快速解决

**Q: Xcode构建失败？**
- 检查证书和描述文件
- 清理构建：Product → Clean Build Folder
- 重启Xcode

**Q: 上传失败？**
- 检查网络连接
- 更新Xcode到最新版本
- 检查Apple系统状态页面

**Q: 构建版本不出现？**
- 等待10-30分钟
- 检查邮件是否有错误通知
- 刷新App Store Connect页面

**Q: 审核被拒绝？**
- 仔细阅读拒绝原因
- 修复所有提到的问题
- 回复审核团队（如需说明）
- 重新提交
