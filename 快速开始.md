# 🚀 5分钟快速发布到GitHub

## 📝 准备工作

**需要的账号**（都是免费的）：
- ✅ GitHub账号：[https://github.com/signup](https://github.com/signup)
- ✅ Vercel账号：[https://vercel.com/signup](https://vercel.com/signup)（用GitHub登录）

---

## 🎯 三步发布

### 第1步：创建GitHub仓库（2分钟）

1. **登录GitHub**，点击右上角 **`+`** → **New repository**

2. **填写信息**：
   ```
   Repository name: mystic-arts
   Description: 中国玄学算命应用
   选择: Public（公开）
   ```
   ⚠️ **不要勾选**任何选项（README、.gitignore、License）

3. **点击** `Create repository`
   
4. **复制页面上的命令**（稍后会用到）

---

### 第2步：从Figma Make导出代码（3分钟）

#### 选项A：Figma Make有导出功能

1. 打开你的Figma Make项目
2. 找到 **Export** 或 **Download** 按钮
3. 下载`.zip`文件
4. 解压到桌面

#### 选项B：手动下载（如果没有导出功能）

**告诉我，我会帮你准备完整的代码包！**

---

### 第3步：上传到GitHub（2分钟）

#### 方法A：用GitHub Desktop（最简单）✅

1. **下载GitHub Desktop**
   - Windows/Mac: [https://desktop.github.com](https://desktop.github.com)

2. **打开GitHub Desktop**
   - 登录你的GitHub账号

3. **添加项目**
   - `File` → `Add Local Repository`
   - 选择你解压的项目文件夹
   - 如果提示不是Git仓库，点击 `Create a repository`

4. **发布**
   - 点击 `Publish repository`
   - 名字：`mystic-arts`
   - 取消勾选 `Keep this code private`
   - 点击 `Publish Repository`

**完成！** 🎉

#### 方法B：用命令行（适合有经验的）

```bash
# 1. 打开终端，进入项目文件夹
cd ~/Desktop/mystic-arts

# 2. 初始化Git
git init
git add .
git commit -m "首次提交：Mystic Arts应用"

# 3. 连接GitHub（替换YOUR_USERNAME为你的用户名）
git remote add origin https://github.com/YOUR_USERNAME/mystic-arts.git
git branch -M main
git push -u origin main
```

---

## 🌐 第4步：部署到Vercel（2分钟）

### 为什么要部署？

GitHub只是存储代码，Vercel让你的应用可以被访问：
- ✅ 自动HTTPS（安全）
- ✅ 全球CDN（快）
- ✅ 自动更新（方便）
- ✅ 完美PWA（iOS状态栏正确）

### 开始部署

1. **访问Vercel**
   - [https://vercel.com/signup](https://vercel.com/signup)
   - 点击 **Continue with GitHub**（用GitHub登录）

2. **导入项目**
   - 点击 **Add New...** → **Project**
   - 找到 `mystic-arts`
   - 点击 **Import**

3. **确认设置**（不用改）
   ```
   Framework Preset: Vite  ✅
   Root Directory: ./      ✅
   Build Command: 自动检测   ✅
   Output Directory: dist  ✅
   ```

4. **点击Deploy**
   - 等待1-2分钟
   - 完成！🎉

5. **获取网址**
   - 部署完成后，你会得到：
     ```
     https://mystic-arts.vercel.app
     ```
   - 这就是你的应用网址！

## 🔄 以后怎么更新？

### 用GitHub Desktop（最简单）

1. 修改代码
2. 打开GitHub Desktop
3. 填写更新说明（比如"修复bug"）
4. 点击 `Commit to main`
5. 点击 `Push origin`

**Vercel会自动重新部署**（约30秒）

### 用命令行

```bash
git add .
git commit -m "描述你的更改"
git push
```

## ❓ 常见问题

### Q1: 我没有从Figma Make导出代码怎么办？

**A**: 告诉我，我会：
1. 帮你打包完整的项目代码
2. 提供下载链接
3. 你直接上传到GitHub

### Q2: GitHub Desktop推送时要求密码？

**A**: 在GitHub Desktop中：
1. `File` → `Options` → `Accounts`
2. 点击 `Sign in` 重新登录
3. 会打开浏览器授权

### Q3: Vercel部署失败？

**A**: 检查：
1. GitHub仓库是否有完整的代码
2. 是否有 `package.json` 文件
3. 查看Vercel的错误日志（点击Build标签）

### Q4: 如何绑定自己的域名？

**A**: 
1. Vercel项目 → Settings → Domains
2. 输入你的域名
3. 按提示在域名注册商添加DNS记录

### Q5: 应用可以改名字吗？

**A**: 可以！
1. GitHub: 仓库Settings → 改Repository name
2. Vercel: 项目Settings → 改Project Name
