## ⚡ 命令行版（给懂终端的人）

```bash
# 1. 上传到GitHub
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/你的用户名/mystic-arts-app.git
git push -u origin main

# 2. 部署到Vercel
npm install -g vercel
vercel login
vercel --prod

# 3. 完成！得到网址
```

## 🆘 常见问题速查

| 问题 | 解决方案 |
|------|----------|
| Service Worker未注册 | 确保HTTPS环境 |
| 安装提示不显示 | 清除缓存，等3秒 |
| 图标不显示 | 生成PNG图标 |
| 部署失败 | 查看Vercel日志 |
| 手机打不开 | 检查网址是否HTTPS |
| iPhone不能安装 | 必须用Safari浏览器 |
