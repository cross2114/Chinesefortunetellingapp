# 🚀 从这里开始！（5分钟装到手机）

## 第一步：上传到GitHub（3分钟）

### 最简单方法：用GitHub Desktop

1. **下载GitHub Desktop**
   - 网址：https://desktop.github.com
   - 安装并登录（没账号就注册一个）

2. **添加项目**
   - 打开GitHub Desktop
   - 点 File → Add Local Repository
   - 选择你的项目文件夹
   - 如果提示不是Git仓库，点"Create Repository"

3. **上传**
   - 点"Publish repository"
   - 点"Publish repository"确认
   - 完成！

---

## 第二步：部署到Vercel（2分钟）

1. **打开Vercel**
   - 网址：https://vercel.com
   - 用GitHub账号登录

2. **导入项目**
   - 点"Add New" → "Project"
   - 找到你刚上传的项目
   - 点"Import"

3. **部署**
   - 直接点"Deploy"
   - 等2-3分钟
   - 复制你的网址（比如：https://xxx.vercel.app）

---

## 第三步：装到手机（1分钟）

### 📱 安卓手机
1. 用Chrome打开你的网址
2. 等3秒，点底部的"INSTALL APP"
3. 完成！

### 🍎 苹果手机
1. 用Safari打开你的网址
2. 点底部的分享按钮📤
3. 点"添加到主屏幕"
4. 完成！

---

## 🎉 完成！

你的应用现在可以：
- ✅ 像真正的APP一样在手机上运行
- ✅ 离线使用
- ✅ 自动更新

---

## 📸 缺少图标？

应用需要图标才能正常安装：

1. **访问**：https://www.pwabuilder.com/imageGenerator
2. **上传**：`public/icons/icon.svg`（在你的项目里）
3. **下载生成的图标**
4. **放到**：`public/icons/` 文件夹
5. **重新上传**：
   - GitHub Desktop → 写个提交信息 → Commit → Push
   - Vercel自动重新部署
